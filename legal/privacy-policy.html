Privacy Policy for GyanGo

Last updated: 2026-04-20
App name: GyanGo
Package name: ai.gyango
Publisher: Gyango

1) Overview

GyanGo is an offline-first AI learning assistant. We design the app to process user interactions on-device wherever possible.

This Privacy Policy explains what information may be accessed or processed when you use GyanGo, including optional microphone, camera, and photo-selection features.

2) Permissions and user control

GyanGo requests sensitive permissions only when needed and after a user action.

  • Microphone (android.permission.RECORD_AUDIO): used only when the user chooses voice input (for example, tapping a mic button to ask a question). The user can deny microphone permission and continue using text input.
  • Camera / Photos (when enabled in feature flows): used only when the user chooses to capture an image or select one for analysis. The user can cancel image capture/selection at any time.

You can revoke permissions at any time from Android Settings.

3) What data is processed

Depending on what the user chooses to do in the app, GyanGo may process:

  • text prompts entered by the user,
  • voice audio for speech-to-text interactions,
  • images selected or captured by the user for question analysis,
  • basic app diagnostic information (for reliability and troubleshooting on the device).

4) How data is used

  • generate AI responses to learning questions,
  • convert speech to text when voice mode is used,
  • extract text or context from selected images,
  • improve app stability and safety behavior.

5) Data sharing and selling

Gyango does not sell personal data collected through GyanGo. For core offline-first features, processing is designed to occur on-device. If future optional cloud-connected features are introduced, this policy and Play Console disclosures will be updated before rollout.

6) Data retention

Retention depends on feature usage and local app behavior:

  • interaction data is primarily handled on-device,
  • users can clear app data through Android system settings,
  • data that is no longer needed for functionality should not be retained longer than necessary.

7) Children and safety

GyanGo is designed as a kids-first educational assistant with safety guardrails. Parents/guardians should supervise younger users and review device-level permission controls.

8) Security

We use reasonable technical and organizational measures to protect app data. No method of transmission or storage is 100% secure, but we continuously improve safeguards.

9) Changes to this Privacy Policy

We may update this Privacy Policy as features evolve. The "Last updated" date reflects the latest version.

10) Contact

For privacy questions or requests, contact: privacy@gyango.ai
